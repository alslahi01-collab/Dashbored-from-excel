# Smart Data Analyzer (Arabic) 📊

موقع يعمل بالذكاء الاصطناعي لتحليل ملفات الإكسل، توحيد المسميات المتشابهة تلقائياً، وتوليد تقارير Word ولوحات معلومات تفاعلية.

## المميزات:
- توحيد مسميات القرى والمدن (اختيار الأكثر تكراراً أو الأقصر طولاً).
- إنشاء رسوم بيانية تلقائية.
- كتابة تعليقات تفسيرية باللغة العربية.
- تصدير تقرير Word منسق جاهز للطباعة.
